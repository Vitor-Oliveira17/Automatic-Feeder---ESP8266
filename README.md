# Automatic-Feeder---ESP8266
An application to control a custom built cat food dispenser
(U+25CF) Tracks the amount of food left in storage via an Ultrasonic Sensor HC-SR04
(U+25CF) Dispenses food rotating a Tower Pro SG90 9g Micro Servo Motor
(U+25CF) Webpage shows the amount of food in storage
(U+25CF) Configurable time and amount of food dispense via the webpage
